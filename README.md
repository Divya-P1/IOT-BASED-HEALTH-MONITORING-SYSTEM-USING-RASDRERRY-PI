IoT-Based Health Monitoring System using Raspberry Pi

Overview
This project is an IoT-based healthcare system that monitors vital health parameters such as heart rate, SpO₂, ECG, body temperature, and fall detection using Raspberry Pi. Sensor data is processed in real time and can be accessed remotely for continuous patient monitoring.

Features
- Real-time health monitoring
- Heart rate monitoring
- SpO₂ monitoring
- ECG monitoring
- Temperature monitoring
- Fall detection
- Cloud-based remote monitoring

Technologies Used
- Raspberry Pi 4
- Python
- IoT
- SPI Communication
- MCP3208 ADC
- ECG Sensor
- Pulse Sensor
